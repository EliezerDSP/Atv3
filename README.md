# Atividade Ollama Embeddings

Este projeto demonstra a instalação local do Ollama e o uso de Python para gerar vetores de embeddings.

## Como Reproduzir
1. Instale o Ollama em [ollama.com](https://ollama.com).
2. Baixe o modelo: `ollama pull nomic-embed-text`.
3. Instale as dependências: `pip install -r requirements.txt`.
4. Execute o script: `python main.py`.
